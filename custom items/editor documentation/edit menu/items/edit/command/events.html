Custom items can be configured to execute commands when certain events happen. This page lists all events that you can use. Note that you can scroll down to see more events. It should look like this:

  • Clicking the 'Cancel' button will take you back to the item edit menu and discard all changes you made to the commands.
  • Clicking the 'Done' button will take you back to the item edit menu and keep all changes you made to the commands.

Each event has a display name (top left), description (bottom) and a Commands... button (top right). Clicking the button will take you to the command list, where you can configure the commands for the corresponding event.
