The replace blocks menu can be used to configure which block(s) can be replaced by a (custom) ore vein generator. It can also be used to configure on which block(s) custom trees can grow. It should look like this:

You can select both vanilla blocks and custom blocks.

Replacing blocks

When configuring the replacable blocks of a custom ore vein, each of the blocks you add here can be replaced by an ore block of the ore vein generator. In most cases, you should only choose Stone, but you are free to choose any block(s) you want.

Allowed tree terrain

When you are configuring the terrain on which a custom tree can grow, the trees can grow on any of the blocks you select here. For regular trees, you should usually choose just Dirt and Grass block.
