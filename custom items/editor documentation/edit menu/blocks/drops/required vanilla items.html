The required vanilla items menu can be used to configure the list of required vanilla minecraft items of a custom block drop. If you allow only a stone pickaxe, it should look like this:

On the left, there is

  • a Cancel button. Clicking it will take you back to the required items menu, without updating the list of vanilla items.
  • an Add new button. Clicking it will add a new default entry to the list of items on the right. (Afterwards, you should change the values of that entry to your liking.)
  • an Apply button. Clicking it will take you back to the required items menu and update the list of vanilla items.

The list of actual items is shown on the middle/right. For each item/entry, there is

  • an item button. The text on the item button shows the currently selected item. Click the button to select another item.
  • an Allow custom items checkbox. This determines whether custom items that have the chosen vanilla item type as Internal item type are also allowed. (If you keep the checkbox checked, both vanilla stone pickaxes and custom stone pickaxes can be used. If you uncheck it, only vanilla stone pickaxes can be used.)
  • a red cross on the right. Clicking that will remove the item/entry.
