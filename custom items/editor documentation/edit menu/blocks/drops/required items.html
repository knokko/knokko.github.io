The required items menu can be used to configure which items must be used to break a custom block in order to let it drop items. If you enable it (by clicking the checkbox), it should look like this:

In this menu, you can configure a list of custom items and a list of vanilla minecraft items. In order to get the block drops, players will have to use either a custom item on the list or a vanilla minecraft item on the list.

  • The Required custom items is the list of custom items. Click the Change... button to change the list.
  • The Required vanilla items is the list of vanilla minecraft items. Click the Change... button to change the list. See this page for more information.
  • If you check Invert, the meaning of these required items will be inverted. If you leave it unchecked, these required items act like a whitelist: any item that is on the list of vanilla items or custom items can be used. If you check Invert, these required items will act like a blacklist instead: any item can be used to break the block, except the items on the list of vanilla items or custom items.
