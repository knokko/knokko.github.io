The manual output slot menu can be used to add manual output slots to a custom container. Manual output slots are the slots in which the resulting items of container recipes appear will appear as soon as the inputs are placed. The inputs will be consumed when the player clicks on the manual output slot. (So it works like the result slot of a crafting table.) Just like regular output slots, manual output slots have only a name and a placeholder. It should look like this:

  • The 'Cancel' button will take you back to the container edit menu you came from, without adding a new manual output slot.
  • The 'Done' button will replace the original slot with a new manual output slot with the provided name and placeholder. It will then take you back to the container edit menu you came from.
  • You should type the name of the manual output slot you're about to create in the edit field on the right of 'Name:'. No two manual output slots of the same custom container can have the same name. The name will not be used in-game, but it will be used to link the manual outputs of the container recipes to the manual output slots. So to move a manual output slot, you would delete it and create a new manual output slot with the same name somewhere else.
  • The Placeholder will be displayed in the slot when no manual container recipe is ready. (For instance when the player hasn't placed any ingredients yet.) Click the Choose... button to choose the placeholder and click the Clear... button to remove it. Note: the placeholder is optional!
