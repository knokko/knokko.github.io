The output table entry menu can be used to add entries to output tables. Such an entry is a pair of a result item and a chance that the result item is chosen. This menu should look like this:

  • The 'Cancel' button will take you back to the output table you came from, without adding the new entry.
  • The 'Create' button will add the entry to the output table you are editing or creating, and take you back to its menu.
  • The 'Chance' edit field lets you choose the chance that this entry will be chosen (in percentages, must be an integer).
  • The 'Choose...' button on the right of 'Item:' will take you to the choose result menu, where you can choose the item of this entry.
