The item type selection menu is the root menu from which you can add items as ingredients to crafting recipes. It should look like this:

The menu consists out of the type buttons on the right and a back button on the left. In the middle is a description of the currently selected ingredient (possibly 'empty').

The 'Back' button will take you back to the previous menu (being Create Shapeless Recipe or Create Shaped Recipe).

The item type buttons, open up a submenu where you can select an item of that type. For more information, visit the respective pages:
