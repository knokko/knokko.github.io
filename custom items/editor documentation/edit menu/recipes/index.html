This plug-in provides several types of custom recipes:

  • Crafting recipes (shaped + shapeless)
  • Cooking recipes ((blast) furnace + smoker + campfire)
  • Smithing recipes

Furthermore, it provides so-called upgrades that can be used to create recipes that modify ingredients.

Minecraft versions

  • Crafting recipes are supported in any MC version that is supported by this plug-in.
  • Furnace recipes are only supported on MC 1.13 and later because the Bukkit API became much better in MC 1.13.
  • The other cooking recipes are only supported on MC 1.14 and later because the corresponding blocks were added in MC 1.14.
  • Smithing recipes are only supported on MC 1.20 and later because the smithing recipe format of minecraft changed in MC 1.20. This plug-in only supports the new format.
