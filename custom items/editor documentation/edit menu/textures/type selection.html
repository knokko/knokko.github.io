In the texture type selection, you can choose what kind of texture you want to load. It should look like this:

Currently, there are 2 types of textures: normal textures and bow textures. As the name suggests, bow textures are for bows. Normal textures are for all other items. As you can see, each type has its own button:

  • The 'Load simple texture' button will bring you to the texture edit menu where you can choose a name and select an image.
  • The 'Load bow texture' button will bring you to the bow texture edit menu where you can choose a name and image, but also select images for the pulling textures.
Finally, there is the 'Back' button that will bring you back to the texture overview without loading a new texture.
