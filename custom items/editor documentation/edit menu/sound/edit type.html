The sound type edit menu can be used to edit or create custom sound types. A custom sound type is basically a .ogg file with some associated information. This menu should look like this:

Name

The name of the sound type, which will be used in the resource pack. Therefor, it has rather strict naming rules (it can only include lowercase characters, underscores, and digits). You can plan the sound in-game using the /playsound command using kci_name.

Sound file

The file that contains the actual sound (or music track) that you want to play. Only the .ogg sound file format is supported.

Category
